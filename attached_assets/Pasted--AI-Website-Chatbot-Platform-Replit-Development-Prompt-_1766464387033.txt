# AI Website Chatbot Platform - Replit Development Prompt

You are an expert full-stack developer specializing in modern web applications. Build a complete AI website chatbot SaaS platform with the following specifications:

## Tech Stack Requirements
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4 + shadcn/ui components
- **State Management**: Zustand
- **UI Components**: shadcn/ui with custom animations
- **Icons**: Lucide React
- **Typography**: Inter font family
- **Animations**: Framer Motion for smooth interactions

## Design Philosophy
- **Modern, premium aesthetic** - NOT boring flat design
- **Glassmorphism effects** with subtle gradients and backdrop blur
- **Smooth micro-interactions** - hover effects, transitions, loading states
- **3D depth** using shadows, layers, and perspective transforms
- **Animated gradients** in backgrounds and CTAs
- **Neumorphism elements** for cards and containers
- **Dark mode by default** with elegant light mode support
- **Bento grid layouts** for feature sections
- **Floating elements** with subtle parallax effects

## Color Palette
- Primary: Vibrant purple gradient (#8B5CF6 to #6366F1)
- Secondary: Cyan accent (#06B6D4)
- Background Dark: Deep navy (#0A0E27) with subtle grain texture
- Background Light: Soft gray (#F8FAFC)
- Success: Emerald (#10B981)
- Warning: Amber (#F59E0B)
- Error: Rose (#F43F5E)

## Functionality Requirements

### Backend Integration
```typescript
// Supabase setup
- User authentication (email/password + OAuth)
- Database schema:
  - users table
  - chatbots table (user_id, name, url, settings, created_at)
  - conversations table (chatbot_id, messages[], created_at)
  - embeddings table (chatbot_id, content, vector)

// OpenAI Integration
- GPT-4 for chat responses
- text-embedding-3-small for vector embeddings
- Streaming responses support

// Web Scraping
- Cheerio for HTML parsing
- Extract text content from pages
- Handle pagination (max 50 pages)
- Sitemap.xml parsing support
```

### State Management (Zustand)
```typescript
// chatbot-store.ts
interface ChatbotStore {
  chatbots: Chatbot[]
  currentChatbot: Chatbot | null
  isLoading: boolean
  fetchChatbots: () => Promise<void>
  createChatbot: (data: CreateChatbotData) => Promise<void>
  updateChatbot: (id: string, data: Partial<Chatbot>) => Promise<void>
  deleteChatbot: (id: string) => Promise<void>
}
```

## Installation Instructions
```bash
# Initialize Replit project with Next.js 15
npx create-next-app@latest . --typescript --tailwind --app

# Install dependencies
npm install @supabase/supabase-js openai zustand cheerio
npm install framer-motion lucide-react
npx shadcn-ui@latest init
npx shadcn-ui@latest add button card input textarea select dropdown-menu dialog toast

# Environment variables needed (.env.local)
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
OPENAI_API_KEY=
NEXT_PUBLIC_APP_URL=
```
